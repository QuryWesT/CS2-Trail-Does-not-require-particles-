The purpose of this plugin is to create a trail effect behind the player using specific codes. The best part is that it does not require any effects, particles, or models. It works by using the gameâ€™s built-in color codes, providing an FPS-friendly and fun experience. Additionally, you can easily add or remove new colors through the configuration file.

Configs; 
<details>
<pre>
{
  "Colors": {
    "blue": "0,0,255",
    "red": "255,0,0",
    "green": "0,255,0",
    "yellow": "255,255,0",
    "purple": "128,0,128",
    "cyan": "0,255,255"
  },
  "BeamWidth": 2,
  "BeamLife": 0.5,
  "ConfigVersion": 1
}
</pre>
</details>

Say Command;
<details>
<pre>
!trail <color|off>
!trail off   // Disables the trail
!trail       // Displays the available colors
</color>
